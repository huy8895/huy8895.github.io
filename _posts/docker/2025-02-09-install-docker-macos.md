# Đối với macOS
## 1. Yêu cầu hệ thống
- macOS phiên bản 10.15 (Catalina) hoặc mới hơn
- Chip Intel hoặc Apple Silicon (M1/M2)
- Tối thiểu 4GB RAM (khuyến nghị 8GB trở lên)
